Список команд

turn\ - (бейся!)
turn\ + (подкидывай!)
turn\ масть, величина карты

deck\ кол-во карт в колоде, максимальное кол-во карт в колоде
trmp\ масть, величина карты
gamt\ масть, величина, \, масть, величина, масть, величина, ...
	"\" - символ для разделения карты, которой отбились
wint\ игрок, игрок, игрок, ... (в порядке выхода из игры - таблица победителей
		текущей игры)
your\ масть, величина, ... (карты игрока)

mesg\ текст
lspl\ №меня, №кто ходит, №кто отбивается / имя игрока, кол-во карт, ...
scor\ имя игрока, кол-во побед, игрок, кол-во побед, ...
exit\

take\ (сигнал о том, что игрок хочет затянуть карты)
take\ имя_игрока (рассылка остальным, что игрок с таким именем берет карты)

<card> = <value><color>
<value> = значение карты
<color> = масть карты
<player> = имя игрока

ход:
turn/[+|-|[<card>|no]]
+ = пердложить подкинуть карту
- = предложить отбиться
<card> = возвращаем сделанный ход
no = нет хода.

колода:
deck/[n,m]
n = количество карт в колоде
m = полный размер колоды

козырь:
trmp/[<card>]
<card> = козырь

Игровой стол:
gamt/[<stack>/.../<stack>]
<stack> = <card>,...,<card>,

таблица победивших в данной игре:
wint/[<player>,]...[<player>,]
карты игрока:
your/[<card>,]...[<card>,]

сообщение:
mesg/<text>
<text> = текст сообщения

список игроков:
lspl/a,b,c/[<player>,n,...,<player>,n,]
a = индекс игрока в этом списке
b = индекс подбрасывающего игрока
c = индеккс отбивающегося игрока
n = количество карт у конкретного игрока

счёт:
scor/[<player>,n,...,<player>,n]
n = количество очков/побед у каждого игрока

отключение от сервера:
exit/
