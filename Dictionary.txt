Список команд

"/", "," = разделители внутри команд

<card> = <value><color>
<value> = значение карты
<color> = масть карты
<player> = имя игрока

Отправка имени игрока:
name/[<player>]
name/ = запрос об имени игрока

ход:

turn/[+|-|[<card>|no]]

+ = предложить подкинуть карту

- = предложить отбиться

<card> = возвращаем сделанный ход

no = нет хода.



колода:

deck/[n,m]

n = количество карт в колоде

m = полный размер колоды



козырь:

trmp/[<card>]

<card> = козырь



Игровой стол:

gamt/[<stack>/.../<stack>]

<stack> = <card>,...,<card>,



таблица победивших в данной игре:

wint/[<player>,]...[<player>,]


карты игрока:

your/[<card>,]...[<card>,]



сообщение:

mesg[/<player>]/<text>

<text> = текст сообщения


<player> = имя игрока отправившего сообщение

список игроков:

lspl/k,l,m,[a|p|t]/[<player>,n,...,<player>,n,]

k = индекс игрока в этом списке

l = индекс подбрасывающего игрока

m = индеккс отбивающегося игрока

n = количество карт у конкретного игрока
a = active ходит активный игрок, т.е. подкидывает (атакует)
p = passive ходит отбивающийся игрок
t = take ходит подкидывающий игрок, т.к. отбивающийся затянул



счёт:

scor/[<player>,n,...,<player>,n]

n = количество очков/побед у каждого игрока



отключение от сервера:

exit/
